About Nabla

Overview

Nabla is an intuitive user interface for OpenFoam, designed to showcase the potential of a fully functional UI for running and managing OpenFoam simulations. With Nabla, users can perform a wide range of operations, including:

  • Storing simulations in a database
  • Creating and saving custom simulation dictionaries in a database
  • Validating dictionaries for OpenFoam compatibility
  • Executing simulations
  • Preserving and managing simulation results
